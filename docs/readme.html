PackBSP 2.0.4

Released: Mar 22, 2011

Homepage: http://technofovea.com/blog/projects/packbsp

What it is:

PackBSP is a tool to help automatically manage and pack dependencies in custom maps for games based on the Source Engine by Valve Software. Please note that games which do not use the central Source SDK (Such as Left 4 Dead or Alien Swarm) are not yet supported.

What you need:

  1. You need to have Java 5.0 or higher in order to run this program. (You can get it from www.java.com )
  2. You need to be logged on to Steam so that some of Valve's SDK tools are operable.

What to click:

To run the GUI double-click on packbsp.exe.

"Help! It shows me a list of files it expects, do I really need to tell it where to find them all?"

To some extent you need to use your own judgment whether a missing asset is important. For example, .PHY files are only needed for solid props.

Having problems?

First, check out the support page at:
http://technofovea.com/blog/projects/packbsp/support

If you wish to report problems, please send a description of what went wrong, along with a copy of the packbsp_debug.log file, to:
packbsp@technofovea.com
